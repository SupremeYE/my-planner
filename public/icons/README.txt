PWA / 파비콘 / iOS 홈 화면 아이콘 (파도 Pado 등)

지금은 플레이스홀더 PNG가 들어 있습니다. 피그마·캔바에서보낸 최종본으로 같은 파일명을 덮어쓰면 됩니다.

  (탭은 루트 /favicon.svg 권장. PNG 탭용은 index.html에 link 추가 후 32×32 PNG 배치)
  apple-touch-icon.png  … iOS Safari “홈 화면에 추가” (180×180 또는 512 권장, 캐릭터형 아이콘)
  icon-192x192.png       … PWA 설치·안드로이드 (필수급)
  icon-512x512.png       … PWA 스플래시·고해상도 (필수급)
  icon-72~384            … manifest용 (같은 그림을 여러 크기로 export 해 덮어써도 됨)

캘린더+P 심볼형 / 파도 머리형 등 디자인 2종이 있으면:
  - 탭·작은 곳 → 캐릭터 얼굴 크롭
  - 홈 화면·설치 아이콘 → 스쿼클 안전여백 있는 정사각 아이콘

maskable 용도는 가장자리 10% 여백을 두세요.
